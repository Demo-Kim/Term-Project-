  ***

  각 플레이어는 1~5번으로, 그 중 사용자는 5번 플레이어이다.
  
  플레이어는 '두부 몇 모!'를 외치는데, 두부는 '한 모'부터 '다섯 모'까지 있으며 플레이어 번호와 동일함. 
 
  ex) 1번 - 한 모, 5번(사용자) - 다섯 모 
          
  자기 자신을 외치면 지는 게임이다.
